# SALES DATA ANALYSIS DASHBOARD

## Project Overview
This project analyzes sales data to identify business trends, customer behavior, and sales performance. The analysis helps businesses make data-driven decisions by examining sales patterns and key performance indicators.

## Objectives
- Analyze total sales and revenue.
- Identify top-selling products.
- Find monthly and yearly sales trends.
- Analyze customer purchasing behavior.
- Visualize important business insights.

## Tools Used
- Microsoft Excel
- Python
- Pandas
- NumPy
- Matplotlib

## Project Steps
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insights

## Key Insights
- Identified the best-performing products.
- Found seasonal sales trends.
- Analyzed revenue growth patterns.
- Created visual dashboards for easy understanding.

## Files Included
- Sales_Data.xlsx
- Sales_Analysis.ipynb
- README.md
- Charts and Visualizations

## Conclusion
This project demonstrates skills in data cleaning, data analysis, visualization, and deriving business insights from raw data.
